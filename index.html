Blend-Cleaver-Mini

A modular fusion enzyme that lands on plastic and teams a mini-PETase with a broad polyester esterase to depolymerize PET blends and adhesive-laminates.

Project Overview

Using AlphaFold, I designed a modular fusion enzyme to break down PET blends and adhesive-laminates.

Features

  • Modular design allows for easy modification of the enzyme.
  • Ready to be used in a lab setting.
  • Can be used to break down PET blends and adhesive-laminates.
  • Targets multiple types of plastic, using its dual catalysis

What I learned

  • How to use AlphaFold to design an enzyme.
  • How to use the Protein Data Bank to find relevant enzymes.
  • Building a protein structure from scratch.
  • How to read pLDDT, pTM, PAE.
